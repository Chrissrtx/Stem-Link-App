# Stem Link App

## Descripción del Proyecto
Stem Link es una plataforma desarrollada para el curso **CS 2031 - Desarrollo Basado en Plataformas (DBP)**. El objetivo principal es crear una aplicación funcional que integre conocimientos de backend, frontend, bases de datos, testing y despliegue.

## El Problema y la Solución
### El Problema
(Pendiente de definir específicamente según las necesidades de los estudiantes de STEM)

### La Solución
Una plataforma web que conecta a estudiantes y profesionales en áreas de Ciencia, Tecnología, Ingeniería y Matemáticas (STEM), facilitando el intercambio de recursos, mentorías y colaboración en proyectos.

## Funcionalidades Principales
- **Registro de Usuarios:** Perfiles para estudiantes y profesionales.
- **Búsqueda de Mentores/Colaboradores:** Filtrado por área de especialidad y habilidades.
- **Gestión de Recursos:** Espacio para compartir y acceder a material educativo.
- **Chat Integrado:** Comunicación directa entre usuarios para coordinación de proyectos o mentorías.
- **Sistema de Reseñas:** Calificación y retroalimentación sobre mentorías y colaboraciones.

## MVP (Mínimo Producto Viable)
- Registro y autenticación de usuarios.
- Búsqueda básica de perfiles.
- Sistema de mensajería interna.

## Aspectos Técnicos
- **Frontend:** React (TypeScript) / Angular (según estándares de DBP).
- **Backend:** Node.js (Express) / Python (FastAPI).
- **Base de Datos:** Relacional (PostgreSQL/MySQL) para manejo de usuarios, perfiles y relaciones.
- **Servicios Externos:** (Ej: Google Auth para registro, AWS/Cloudinary para almacenamiento de archivos).

- **Semana 10:** Entrega de Backend Completo (Domingo 24 de Mayo).
- **Final del Curso:** Presentación en formato "demo" y despliegue.
